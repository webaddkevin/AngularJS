#Angular JS 即学即用 <small>brad Green著</small>

本书循序渐进地解读了AngularJS，每一章中出现的概念，都会在后续一章节中进行单元测试。

1. 第1章  AngularJS 概述。 介绍AngularJS及涵盖的各种概念
2. 第2章  基本的AngularJS指令及控制器。
3. 第3章  AngularJS中的单元测试。介绍如何通过Karma和Jasmine这两种工具来进行单元测试。
4. 第4章  表单、输入和服务。介绍如何将AngularJS的效力发挥到极致。
5. 第5章  AngularJS服务揭秘。介绍AngularJS中服务的概念及一些内置服务。
6. 第6章  $http与服务器通信。 介绍如何使用$http及高级$http来和服务器进行交互。
7. 服务及XHR的单元测试。本章深入研究如何为服务进行单元测试及$httpBackend模拟服务器请求。
8. 第8章 过滤器的使用
9. 第9章 过滤器测试
10. 第10章 ngRoute与页面迁移。
11. 第11章 指令。介绍了一些关于指令的基本概念以及如何创建指令
12. 第12章 指令的单元测试。本章介绍了如何对指令进行单元测试。
13. 第13章 进阶指令。对高级指令解析
14. 第14章 端对端测试。介绍如何使用Protractor和WebDriver进行端对端测试
15. 第15章 指导方针和推荐做法。本章介绍AngularJS中应当遵守的规约以及一些相当实用的工具
